Documentation B-pay SDK



Bienvenue dans la documentation du SDK de B-pay, La présente documentation est un guide de démarrage rapide visant à vous montrer les différentes étapes pour la création et la configuration d’un projet qui sera associé à votre application ou business.


Vous pouvez utiliser ces SDKs pour accéder à tous nos points de terminaison d'API, tels que l'API Business pour l'utilisation de notre système payement dans un Shop (Supermarché, Alimentation,Etc) ou l'API Développer pour son utilisation sur une plateforme en ligne (site ou application de e-commerce).
